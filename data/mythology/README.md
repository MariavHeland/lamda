# LAMDA Mythology Database

## Purpose

This mythology database gives LAMDA the ability to analyze screenplays through narrative traditions beyond Hollywood's three-act structure. When a screenplay's structure, protagonist logic, or thematic DNA echoes a non-Western tradition, LAMDA can identify it and say:

> "Your story's structure is closer to the Popol Vuh than to the Hero's Journey — here's what that means."

## Files

### 1. `narrative-structures.md` (73K chars)
**The core structural logic of 16+ world traditions.**
- How each tradition builds stories (linear, cyclical, spiral, dreamtime, nested)
- How time works in each (Western linear vs Hindu cyclical vs Aboriginal coexisting)
- Protagonist logic (active Western hero vs reluctant Japanese hero vs communal African protagonist vs land-as-character)
- What "resolution" means in each tradition
- Contrasted against Hollywood three-act as baseline

**Traditions covered:** Greek, Norse, Egyptian, Hindu, Japanese, Chinese, Celtic, Mesopotamian, Yoruba/West African, Aztec/Maya, Polynesian, Slavic, Aboriginal Australian, Persian, Korean, Arabian/Islamic

## How LAMDA Should Use This

### For the Research Engine:
- When analyzing a screenplay's **structural pattern**, check it against narrative-structures.md — is this a three-act story, or does it follow kishōtenketsu? A sovereignty cycle? A descent-return?
- When identifying **comp titles**, use the film-mapping to suggest films from the tradition that matches the screenplay's structure (not just Hollywood comps)
- When mapping **archetypes**, go beyond Campbell — is this a trickster protagonist? A communal protagonist? A reluctant hero in the Japanese mode?

### For the Script Doctor:
- When diagnosing "problems" — first check if the screenplay is operating in a non-Western structure where the "problem" is actually a feature (e.g., "lack of conflict" in a kishōtenketsu structure is intentional)
- When suggesting fixes, reference how the relevant tradition handles that narrative moment
- When analyzing protagonist agency, consider whether the screenplay uses Western active-hero logic or another tradition's protagonist model

### Key Insight for LAMDA:
Many screenplays that seem "broken" by Hollywood standards are actually operating in a different narrative tradition. LAMDA's job is to recognize which tradition and analyze the screenplay on its own terms, while still being able to flag genuine structural problems.
